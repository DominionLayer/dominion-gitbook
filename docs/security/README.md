# Security

Security mechanisms in Dominion.

## Overview

Dominion is designed with security as a core principle:

- **Approval gates** — All actions pass through checkpoints
- **Auditor agents** — Independent verification
- **Rate limiting** — Prevent abuse
- **Key management** — Secure credential handling
- **Privacy** — Minimal data collection
